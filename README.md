# API Test Framework - Karate

### Getting Started
```
git clone https://github.com/sadabnepal/karate-api-test.git
cd karate-api-test
```

### Run Test
```
mvn test
```

### View Report
```
Path: {PROJECT_ROOT_FOLDER}/target/karate-reports/karate-summary.html
```
